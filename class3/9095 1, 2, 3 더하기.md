아 이렇게 쓰고 보니까 DP라는거 알 것 같다.

가지를 하나 놓고 쪼개면 그 뒷부분은 전 단계의 수가 된다.

`T(n) = T(n-1) + T (n-2) + T(n-3)`

이걸 거다 초항은 1, 2, 3이고 





```python
# 9095 1,2,3더하기

def A(n):
    if n == 1:
        return 1
    elif n == 2:
        return 2
    elif n == 3:
        return 4
    return A(n-1) + A(n-2) + A(n-3)


T = int(input())
for i in range(T):
    n = int(input())
    result = A(n)
    print(result)
```
